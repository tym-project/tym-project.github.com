Ouvrir Une Invite De Commande Dans Le Dossier Courant (Windows)

May 19th, 2010

Je n’aime pas la ligne de commande Windows, mais sous Windows 7 je n’ai pas trop le choix ! Ce qui m’insupporte encore plus c’est de devoir naviguer dans mon arborescence à coups de cd.
Mon utilisation de Windows étant essentiellement graphique (sans ligne de commande potable…pas trop le choix !), je me trouve souvent dans le dossier dans lequel je veux lancer un traitement avec une commande.

Pour ouvrir une cmd dans le dossier courant, il vous suffit de faire <SHIFT> + Clic droit, et ensuite de choisir “Ouvrir une invite de commande ici” :

Simple…enfin simple !
