Socksifier Une Application Sous Linux

Apr 27th, 2009

Si vous avez suivi cet article, vous savez faire des tunnels ssh sous Linux. Mais que faire quand vos applications n’intègrent pas la fonctionnalité de proxy Socks ?
tsocks va bien vous simplifier la vie : c’est un “socksifieur”, une application qui fait passer tout et n’importe quoi via un proxy Socks. Sous Archlinux, rien de plus simple ! Commencez par installer le package :

# pacman -S tsocks

Le fichier de configuration se trouve dans /etc/tsocks.conf (il faut le créer la première fois) , éditez le et configurez le selon vos besoins. Voici mes paramètres pour un tunnel SSH sur le port 8080 :

# Adresse du serveur
server = 127.0.0.1
# Port du serveur
server_port = 8080
# Type de sereur socks, ssh fait du 5
server_type = 5

Ensuite pour socksifier une application (Pidgin dans l’exemple) :

tsocks pidgin

Bon socks ;-)

Un Peu De Divertissement “Qui Fait Du Bien Par Où ça Passe” !

Apr 18th, 2009

Voici un projet que je trouve vraiment bien : Defend Unreal Territories ! Mais rien de mieux qu’une présentation par l’équipe elle même : _ “Tout d’abord, DUT, qu’est-ce que c’est ?_

Defend Unreal Territories est un nouveau gametype pour Unreal Tournament 3. Le concept s’inspire des Tower Defense comme on peut les voir dans Warcraft. Le but est d’empêcher les hordes de monstres d’atteindre un objectif donné. Dans un premier temps, l’équipe des joueurs luttera donc à pieds, avec des armes conventionnelles. Puis les joueurs pourront débloquer des bonus (de nouvelles capacités, des objets etc.) grâce aux points accumulés à chaque fois qu’un monstre est tué. Grâce aux nouvelles capacités, les joueurs pourront être plus rapides, avoir plus de vie, faire plus de dégâts, etc. Nous prévoyons aussi de les épauler par des tourelles automatiques.

Quand est-ce que ce sera fini ?

Tout dépend de l’avancement de nos travaux… Nous distribuerons sans doute une première version du mod le 15 mai 2009 Pour le moment, c’est presque jouable. Mais pas franchement intéressant, il nous manque encore plein de fonctionnalités. Patience !

Qui êtes-vous ?

_ Une équipe de plusieurs personnes à SUPINFO, l’école supérieur d’informatique. Nous avons décidé, avec l’aval de l’administration, d’intégrer ce projet dans notre cursus._

Le site web du mod est www.defend-ut.com”

Bon jeu ;-)
